About Advanced Search

You can select a different search method by clicking on the different tabs. Default is 'Auto Search'.

You can limit the date periods applied to your search by specifying the 'From' and/or 'To' dates.

You can search multiple databases at once by clicking the database name for each database you want to include in your search.

While not every database is individually listed, every database is included in one or more group listings (eg Commonwealth: All Cases).
